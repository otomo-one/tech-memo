===========================
Apahce
===========================

環境はCentOS 6.3

インストール
--------------------------

::

    # yum -y install httpd

設定ファイル
--------------------

/etc/httpd/conf/httpd.conf
にある。

詳しい設定はあとで追加。


起動
------------------------
デフォルトのドキュメントルートが/var/www/html/になっているので、
そこにindex.htmlなどを準備する。

::

    # /etc/rc.d/init.d/httpd start

起動して当該アドレスにアクセスするとindex.htmlが表示される。
FWでポート80があいてない場合はiptablesであける。
